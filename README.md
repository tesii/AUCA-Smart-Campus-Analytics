### Smart Campus ZK Dashboard

A full-stack web application built with Spring Boot, ZK Framework, and JPA (Hibernate) for managing staff, departments, roles, and requests in an admin dashboard system.

## 🚀 Features
Admin login/logout system
Staff management (Create, Read, Delete)
Department management (Create, Update, Delete)
Role management
Request tracking system

---
## Dashboard statistics:
Total requests
Success / No result / Greeting breakdown
Most & least requested titles
Dynamic ZK UI pages
Session-based authentication

---
## 🏗️ Tech Stack

## Backend
Java 17
Spring Boot 2.7.18
Spring Data JPA
Hibernate
Spring Security Crypto

---
## Frontend (UI)
ZK Framework 9.6.0.2
ZUL pages
MVC (SelectorComposer)
Database
MySQL
